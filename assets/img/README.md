# Images

Drop project/profile images here. Filenames are referenced from `assets/js/projects.js`
(project `thumb` / `gallery`) and from `index.html` (headshot).
Until a file exists, the site shows a clean placeholder — nothing looks broken.

See **../../MISSING_PORTFOLIO_ITEMS.md** for the full checklist.

Filenames the site looks for:
- `headshot.jpg` — About section photo
- `lunar-mission.jpg`, `lunar-staging.png` — Crewed Lunar Mission
- `nasa-jsc.jpg` — NASA JSC visualization tools
- `mayott-uav.jpg` — Mayott heavy-lift UAV
- `collins.jpg` — Collins avionics
- `artemis.jpg` — ERFSEDS Artemis rocket
- `spectre.jpg` — ERPL Spectre rocket
- `attitude-sim.png` — AE 426 attitude dynamics
- `nastran.png` — AE 318 FEA
- `airfoil.jpg` — AE 314/315 experimental aero
- `earth-moon-relay.jpg` — AE 429

Generated (don't delete):
- `og-image.png` — social/link-preview image (1200×630)
- `favicon.svg` — browser tab icon

Tip: keep images ≤ ~300 KB each (export ~1600 px wide) for fast loads.
