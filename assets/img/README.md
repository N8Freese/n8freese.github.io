# Images

Drop project/profile images here, then point the matching `<div class="card__media">`
(or `.about__photo`) in `index.html` to them.

Suggested filenames the placeholders reference:
- `headshot.jpg` — About section photo
- `capstone.jpg` — AE 441 launch vehicle
- `mayott.jpg` — Mayott drone flight controller
- `sad-sim.png` — AE 426 attitude-dynamics plot
- `earth-moon-relay.jpg` — AE 429
- `nastran.png` — AE 318 FEA
- `airfoil.jpg` — AE 314/315 experimental aero

Tip: keep images ≤ ~300 KB each (export at ~1600 px wide) for fast loads.
