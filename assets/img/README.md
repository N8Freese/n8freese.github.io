# Images

Drop project/profile images here. Filenames are referenced from `assets/js/projects.js`
(project `thumb` / `gallery`) and from `index.html` (headshot).
Until a file exists, the site shows a clean placeholder — nothing looks broken.

Every image referenced by the site lives here; the authoritative list is whatever
`projects.js` and `index.html` point at. Video poster frames (`*-poster.jpg`)
also belong here, next to their `assets/video/*.mp4`.

Generated (don't delete):
- `og-image.png` — social/link-preview image (1200×630)
- `favicon.svg` — browser tab icon

Tip: keep images ≤ ~300 KB each (export ~1600 px wide) for fast loads.
